# By Bash – Bashayer Noury Digital Card

A minimal personal portfolio and digital business card for **Bashayer Noury**, Product Manager based in Kuwait 🇰🇼.

## Pages

- **Home** `/` — Profile, skills, social links, and a daily rotating quote
- **Card** `/card` — Digital business card with QR code, contact download (vCard), and share link
- **404** — Custom not-found page

## Features

- Light & dark mode with a "Yin Yang" aesthetic (Wine burgundy in light, monochrome in dark)
- Casino-style slot-machine text animation
- Skeleton loading screens matching each page layout
- Animated smoke canvas background
- 50 daily rotating quotes from 10 authors
- One-tap save contact & share functionality

## Tech

Built with React, TypeScript, Vite, Tailwind CSS, and Framer Motion.
